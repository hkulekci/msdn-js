## Subtraction Assignment Operator (-=) (JavaScript) {subtraction_assignment_operator_equal_javascript_title}

### Introduction 

 Subtracts the value of an expression from the value of a variable and assigns the result to the variable.

### Syntax 

```
result -= expression
```
