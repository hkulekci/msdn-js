## Assignment Operator (=) (JavaScript) {assignment_operator_equal_javascript_title}

### Introduction 

 Assigns a value to a variable.

### Syntax 

```
result = expression
```

#### Remarks 

<div id="languageReferenceRemarksSection" class="section" name="collapseableSection" style="">
  <p xmlns:util="util">
    The = operator behaves like other operators, so expressions that contain it have a value. This means that you can chain assignment operators as follows: <span class="code">j = k = l = 0</span>.
    In this case <span class="code">j</span>, <span class="code">k</span>, and <span class="code">l</span> equal zero.
  </p>
</div>
